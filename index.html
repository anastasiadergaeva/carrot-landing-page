Hypestr

Агрегатор популярного контента из социальных медиа

Hypestr позволяет следить за новостями и трендами по интересующий тематике в социальных медиа. Приложение основано на ключевых словах, которые пользователь выбирает, и создает ленты контента, соответствующие этим ключевым словам. Для каждого источника — Telegram, VK, Youtube, Reddit — пользователь может выставить фильтры по популярности отображаемого контента.

Ярмаркет

Ярмаркет — доступная доставка продуктов в удобное время

Онлайн магазин продуктов с доставкой на дом в небольшом городе. Приложение позволяет сделать заказ в удобное время в том числе тогда, когда другие магазины закрыты. Для доставки доступен весь ассортимент продуктов магазина. Оплата заказа происходит в приложении. Чат с поддержкой позволяет оперативно связаться с продавцом в случае возникновения проблем с заказом.
